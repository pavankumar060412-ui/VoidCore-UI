Void Core

Void Core is a fullscreen, cross-platform terminal emulator and system monitor that looks and feels like a futuristic sci-fi computer interface.

Inspired by the visual effects of TRON Legacy (especially the Board Room sequence), Void Core was created to bring a cinematic, advanced computer experience into a real, usable application.

While maintaining a futuristic appearance, Void Core focuses on functionality, customization, and creating a next-generation user experience inspired by science fiction interfaces.

It might or might not be a futuristic experiment taken too seriously.

Features
Fully featured terminal emulator with tabs, colors, mouse events, and support for terminal-based applications.
Real-time system monitoring:
CPU usage
RAM usage
Swap memory
Processes
Network activity
Support for touch-enabled displays with an on-screen keyboard.
Directory viewer that follows the current working directory.
Advanced customization:
Themes
Keyboard layouts
CSS modifications
Interface customization
Optional sound effects for a complete cinematic hacking experience.

Questions & Answers
How do I get Void Core?

Download the latest release from the official Void Core releases page.

Public release builds may require additional permissions depending on your operating system.

I have a problem!

Check the issue tracker to see if your problem has already been reported. If not, create a new issue with details about your setup and the problem.

Can I customize Void Core?

Yes. Void Core supports themes, keyboard layouts, and interface modifications.

Can Void Core run on Raspberry Pi or ARM devices?

ARM builds may be available depending on the release version.

Building From Source

Clone the repository:

git clone <repository-url>

Install dependencies:

npm install

Start Void Core:

npm run start
Building

Create a production build:

npm run build-linux
npm run build-windows
npm run build-darwin

The finished application will be generated in the distribution folder.
